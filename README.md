# 🛒 Blinkit Grocery Data Dashboard

An interactive grocery sales performance dashboard built entirely using **Microsoft Excel** — tracking item category performance, outlet types, and customer preferences across multiple tiers and sizes.



## 📌 Short Description / Purpose

The **Blinkit Grocery Data Dashboard** provides a visually organized summary of sales, outlet performance, and consumer trends across small, medium, and high-sized outlets in various location tiers. Designed for decision-makers to understand revenue drivers, outlet effectiveness, and product demand — all powered by Excel.



## 🧰 Tech Stack

This dashboard was developed entirely using **Microsoft Excel**, with:

- 📊 **Excel Charts and Pivot Tables** – For data visualization and aggregation  
- 📝 **Excel Formulas** – Used for key metrics like Total Sales, Average Sales, and more  
- 🧹 **Data Cleaning & Sorting** – Structured through Excel sheets and filter controls  
- 🎨 **Dashboard Design** – Built with shapes, slicers, and charts for interactive use  
- 📁 **File Format** – `.xlsx` (No external tools used)



## 🗂️ Data Source

**Source:** `BlinkIT Grocery Data Dashboard With Excel.xlsx`  
The dataset contains:

- Product categories (e.g., Frozen Foods, Canned, Breads, Dairy)  
- Outlet sizes (Small, Medium, High)  
- Location Tiers (Tier 1, 2, 3)  
- Sales values, number of items, average customer ratings  
- Outlet types and establishment year  
- Nutritional category: Low Fat vs Regular



## 🚀 Features / Highlights

### 🔹 Business Problem

Blinkit, a last-minute grocery delivery app, needed a quick, Excel-powered dashboard to analyze outlet performance and product sales patterns across India.

Key questions included:

- Which outlet size and location performs best?  
- Which products have the highest demand?  
- What is the distribution of healthy (low-fat) vs regular food items?  
- How do different outlet types contribute to total sales?

### 🔹 Goal of the Dashboard

To create a no-code, Excel-based interactive dashboard that:

- Helps identify high-performing outlet types and regions  
- Breaks down item-level sales and inventory distribution  
- Reveals customer rating averages across outlet categories  
- Highlights nutritional preference and sales mix

## 📈 Business Impact & Insights

- 🧠 **Product Strategy:** Focus on high-selling categories like Snack Foods & Frozen Foods  
- 🏪 **Outlet Expansion:** Tier 3 and Small outlets show strong performance potential  
- 🍽️ **Health-Oriented Planning:** Track demand for Low Fat vs Regular products  
- 📦 **Inventory Allocation:** Use sales per outlet type to optimize stock  
- 📍 **Location Intelligence:** Helps regional managers plan marketing and supply chain efforts
